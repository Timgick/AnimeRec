# AnimeRec
Модель машинного обучения для поиска аниме на основе запросов пользователя
Основной датасет: https://www.kaggle.com/datasets/abhishekgupta56447/anime-offline-database

AnimeRec — веб‑приложение для поиска аниме на основе пользовательских запросов. Система использует методы обработки естественного языка (NLP) и машинного обучения для нахождения аниме, наиболее соответствующих введённому описанию.

Основные возможности
Поиск по ключевым словам: пользователь вводит запрос (например, «приключения фэнтези магия»), система находит наиболее подходящие аниме.

Ручной ввод запросов: текстовое поле для свободного ввода поисковых фраз.

Загрузка пользовательских данных: возможность загрузить собственный CSV‑файл с данными об аниме для расширения базы поиска.

Отображение результатов: для каждого найденного аниме показываются:

название;

жанр;

краткое описание (первые 200 символов);

показатель сходства (от 0 до 1) между запросом и аниме.

Примеры запросов: готовые кнопки с типовыми запросами для быстрого тестирования системы.

Информационная панель: отображает количество загруженных аниме и список доступных колонок данных.

Технологический стек
Python — основной язык программирования.

Streamlit — библиотека для быстрого создания интерактивного веб‑интерфейса.

Pandas — обработка и анализ данных.

Scikit‑learn — реализация алгоритмов машинного обучения:

TfidfVectorizer — преобразование текста в числовые векторы (TF‑IDF);

cosine_similarity — вычисление степени сходства между векторами.

Принцип работы модели
Предобработка данных: объединение полей title, genre и synopsis в единое текстовое поле combined_features.

Векторизация: преобразование текстовых данных в числовые векторы с помощью TF‑IDF.

Поиск сходства: вычисление косинусного сходства между вектором запроса и векторами всех аниме в базе.

Ранжирование: сортировка результатов по убыванию показателя сходства.

Возврат топ‑K результатов (по умолчанию — 5 лучших совпадений).

Инструкция по запуску
Скачайте данные:

Перейдите на Kaggle: 

Скачайте файл anime_offline_database.csv

Поместите его в ту же папку, где будет лежать скрипт

Установите зависимости:

bash
pip install pandas numpy scikit-learn streamlit
Сохраните код в файл app.py

Запустите приложение:

bash
streamlit run app.py
Откройте приложение в браузере по адресу http://localhost:8501
